    Qu'est-ce que la reconduction ?

    La reconduction crée automatiquement une nouvelle année scolaire en copiant depuis {{ source_annee }} :

    • ✅ Les classes (mêmes noms, mêmes sections)
    • ✅ Les matières et enseignants affectés à chaque classe (modifiable après)
    • ✅ Les élèves déplacés vers leurs nouvelles classes (option)
    • 🔄 Les notes repartent de zéro (chaque année = nouvelle saisie)
    • 🔄 Les bulletins seront à créer pour la nouvelle année

    Vous pouvez ajuster les affectations, changer les classes des élèves et créer les bulletins après la reconduction.
