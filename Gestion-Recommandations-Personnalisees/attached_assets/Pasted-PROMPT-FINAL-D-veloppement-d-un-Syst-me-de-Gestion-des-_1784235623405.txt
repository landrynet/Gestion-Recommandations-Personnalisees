PROMPT FINAL – Développement d'un Système de Gestion des Notes et de Publication des Bulletins
Objectif

Développer une application web professionnelle de gestion des notes et de publication des résultats scolaires pour les établissements de la République Démocratique du Congo.

L'application doit permettre au Préfet des études de gérer toute l'organisation académique, aux enseignants d'encoder les notes et au système de générer automatiquement les bulletins officiels.

Le système doit être conçu autour du bulletin officiel, qui constitue la référence principale de toute l'application.

Je fournirai le modèle officiel du bulletin. Tu devras analyser sa structure avant de commencer le développement.

Technologies obligatoires

Utiliser exclusivement :

Backend : Django (Python)
Frontend : HTML5
CSS3
Bootstrap 5
Bootstrap Icons
JavaScript uniquement lorsque nécessaire
Base de données : SQLite

Le projet devra être facilement migrable vers PostgreSQL.

Interdiction

Ne jamais utiliser le panneau d'administration Django (/admin).

Toute l'administration devra être réalisée à partir d'une interface personnalisée.

Module Modèles de Bulletins

Ce module est le cœur de l'application.

Chaque classe et chaque section possèdent leur propre modèle de bulletin.

Exemples :

7ème Générale
7ème Commerciale
8ème Générale
1ère Scientifique
etc.

Lors de la création d'un modèle, le système affiche le bulletin officiel de la RDC.

Les blocs MAXIMA sont déjà préconfigurés conformément au bulletin officiel.

Ils ne sont jamais modifiables.

Le Préfet ne modifie pas les maxima.

Il choisit uniquement les matières déjà enregistrées dans la base de données et les place dans les emplacements prévus du bulletin.

Exemple :

Bloc MAXIMA 20
Religion
Éducation Civique
Informatique
Anglais

Éducation Physique
Bloc MAXIMA 30
Géographie
Histoire
Sciences
Technologie
Bloc MAXIMA 60
Français
Mathématique

Une fois enregistré, ce modèle devient le bulletin officiel de cette classe.

Module Notes

L'enseignant ne remplit jamais directement le bulletin.

L'enseignant saisit uniquement les notes.

Moyenne
Pourcentage

Les notes sont enregistrées dans la base de données.

les moyennes ;
le pourcentage ;
le classement.

Le bulletin généré doit être identique au modèle officiel fourni.

Le bulletin doit être :

imprimable ;
exportable en PDF.

Développement

Ne développe pas tout le projet d'un seul coup.

Travaille module par module.

Avant de passer au module suivant :

le tester ;

Document de référence

Je fournirai le bulletin officiel de la RDC.

Avant de commencer le développement, analyse ce bulletin afin de comprendre sa structure.

L'ensemble du système devra être construit autour de ce modèle, et non l'inverse.

Ne crée pas une application générique de gestion scolaire. Crée un système spécialisé dans la gestion des notes et la génération des bulletins officiels de la RDC, en respectant fidèlement leur structure et leur logique de fonctionnement.
